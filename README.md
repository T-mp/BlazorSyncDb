# BlazorSyncStorage
Eine Komponente die lokalen und remote Speicher, kombiniert und einheitlich Blazor clientseitig zur Verfügung stellt.

## Ziel
Ein Speicher welcher transparent offline und online Persistenz bietet

## Warnung
Dieses Paket ist, wie Blazor selber, noch in einem experimentellen Stadium und die API wird sich wahrscheinlich ändern.

## Warning
This library like Blazor itself is experimental and API is likely to change.
